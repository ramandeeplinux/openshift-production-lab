# OpenShift Virtualization (KubeVirt)
## Objective

This project demonstrates the deployment and management of virtual machines using OpenShift Virtualization (KubeVirt).

The implementation includes:

- Installing OpenShift Virtualization
- Deploying Fedora Virtual Machine
- Bridge Networking
- Static IP Configuration
- Live Migration
- VM Console Access
- Storage Integration with TrueNAS CSI
- VM Verification
