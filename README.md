# Red Hat OpenShift Enterprise Lab

A production-style Red Hat OpenShift laboratory built on VMware vSphere for learning, validating, and documenting enterprise platform engineering use cases.

This repository contains hands-on implementations covering OpenShift installation, storage, virtualization, GitOps, monitoring, AI/ML, service mesh, and Day-2 platform operations.

---

# Technology Stack

- Red Hat OpenShift 4.21
- Kubernetes
- VMware vSphere ESXi
- OpenShift Virtualization (KubeVirt)
- TrueNAS CSI (NFS)
- TrueNAS CSI (iSCSI)
- OpenShift GitOps (Argo CD)
- OpenShift Monitoring
- OpenShift AI
- KServe
- RHEL AI
- Service Mesh (Istio)
- Dynatrace
