# OpenShift 4.x Production Lab

## Overview

This repository contains my enterprise OpenShift laboratory built on VMware ESXi.

The objective of this project is to demonstrate practical experience in deploying, administering and troubleshooting Red Hat OpenShift Container Platform.

## Environment

| Component | Details |
|----------|---------|
| Platform | VMware ESXi |
| OpenShift Version | 4.21 |
| Masters | 3 |
| Workers | 6 |
| Storage | TrueNAS CSI |
| Monitoring | Prometheus + Grafana |
| GitOps | ArgoCD |
| Virtualization | OpenShift Virtualization (KubeVirt) |
