# Day-2 Operations

## Objective

This repository documents real-world Day-2 operational activities performed on a production-style Red Hat OpenShift environment running on VMware vSphere.

The objective is to demonstrate platform administration, lifecycle management, storage operations, troubleshooting, monitoring, virtualization, and operational best practices beyond the initial cluster installation.
